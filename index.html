Buku Panduan Eksekutif

Dokumentasi & Standar Operasional NAVASENA.
HUKUM MUTLAK: UPLOAD SEBELUM LOGOUT!

Seluruh pekerjaan input data (Transaksi, Iuran, Anggota Baru) HANYA TERSIMPAN DI HP ANDA (Offline). Agar data tidak hilang atau menimpa pekerjaan orang lain, Anda WAJIB menekan tombol "UPLOAD" (Ikon Awan Panah Ke Atas) di pojok kanan atas layar setiap kali selesai menginput data!

MENU BERANDA

Fungsi Utama: Dasbor kendali dan eksekusi transaksi.

  • Tombol (+) Melayang: Jantung aplikasi. Gunakan untuk mencatat Pemasukan, Pengeluaran, Uang Muka, dan Dana Talangan.
  • Kirim Laporan WA: Tombol hijau di bawah ringkasan kas untuk melempar ringkasan Pemasukan & pengeluaran bulan ini ke Grup WhatsApp Karang Taruna.
MENU MUTASI

Fungsi Utama: Mencari mutasi lama, melakukan revisi (Koreksi), dan menyelesaikan siklus Uang Muka.

  • Penyelesaian Uang Muka: Contoh: Panitia 17-an membawa kas Rp 500.000. Selesai acara, nota aslinya hanya Rp 400.000. Cari transaksi tersebut, tekan Selesaikan Uang Muka, ketik Rp 400.000. Sisa Rp 100.000 akan otomatis dikembalikan ke Kas Utama.
  • Salah Input (Batal): Cari mutasi yang salah ketik, lalu tekan Hapus Transaksi (Koreksi). Saldo akan otomatis ter-rollback dan Anda bisa membuat transaksi baru yang benar.
MENU ANGGOTA

Fungsi Utama: Sentral Database, Matriks Iuran Bulanan, dan Gamifikasi.

  • Kelola Data Anggota: Input Nama Lengkap, No. WhatsApp, Tgl Lahir, dan Gol. Darah. Data ini wajib akurat untuk fitur KTA dan Broadcast Ulang Tahun otomatis.
  • Matriks Eksekusi Iuran: Ketuk nama anggota, lalu ketuk nama bulan pada grid. Sistem otomatis menghitung XP dan mengamankan saldo. Jika membatalkan pelunasan menyebabkan kas defisit, sistem akan memblokirnya secara mutlak.
  • Auto-Tagihan Masal: Sistem menyapu bersih database untuk mengalkulasi utang dan meluncurkan template WhatsApp penagihan satu per satu tanpa ketik manual.
MENU PROGRAM

Fungsi Utama: Memisahkan pembukuan acara khusus dari kas reguler.

  • Program Baru: Membuat wadah anggaran acara baru (Contoh: "Renovasi Poskamling").
  • Selesai (Arsipkan): Jika program sudah selesai, tekan tombol ini agar acara dipindah ke "Riwayat Arsip" dan tidak memenuhi layar Beranda utama.
MENU SISTEM

Fungsi Utama: Konfigurasi parameter inti, keamanan, dan output dokumen LPJ.

  • Pengaturan: Mengubah tarif iuran bulanan, menambah "Dompet Kas" baru, meracik teks kategori, dan mengubah PIN Login Bendahara.
  • Kotak Pesan: Membaca keluhan, kritik, atau usulan rahasia yang dikirim oleh Warga secara anonim.
  • Load & Backup Excel: Menarik data mentah dari Cloud, atau mengunduh seluruh data menjadi file Spreadsheet (.xlsx).
  • Cetak LPJ: Men-generate Laporan Pertanggungjawaban berbasis AI yang siap diprint (PDF).
  • Tutup Buku Akhir Tahun: Mengompres seluruh mutasi tahun lalu dan menjadikannya "Saldo Awal" di tahun baru.
  • Keranjang Sampah: Memulihkan (Restore) transaksi yang tidak sengaja Anda batalkan.
WAJIB: TARIK DATA CLOUD TERBARU

Jika Anda membuka aplikasi ini dari HP Pribadi, Anda WAJIB menekan tombol "LOAD" (Ikon Awan ke Bawah di sudut kanan atas) sebelum mulai mengawasi data. Ini memastikan layar Anda menampilkan mutasi paling real-time dari server pusat.

(Catatan: Dilarang keras menekan tombol ini jika Anda sedang meminjam HP Bendahara secara langsung).

MENU BERANDA

Fungsi: Pusat informasi agregat kekayaan Karang Taruna saat ini.

  • Total Saldo Tersedia: Angka mutlak dari gabungan seluruh uang kas.
  • Distribusi Dompet: Rincian letak uang (Kas Utama, Dana Darurat, dsb).
  • Ringkasan Arus Kas: Akumulasi Dana Masuk, Dana Keluar, dan Uang Muka/Talangan yang sedang berjalan.
MENU MUTASI

Fungsi: Mengawasi setiap rupiah yang bergerak dengan transparansi total.

  • Menampilkan daftar riwayat transaksi lengkap.
  • Dilengkapi dengan Filter (Semua Mutasi / Mutasi Talangan), kolom pencarian pintar, dan lencana Verified Hash Time-Stamp (Proteksi Blockchain Hijau) pada setiap kartu mutasinya untuk menjamin keaslian waktu pencatatan.
MENU ANGGOTA

Fungsi: Pemantauan Profil, Kepatuhan (Heatmap), & Gamifikasi (Fire Streak).

  • KTA Holografik: Ketuk ikon perisai/pangkat di sebelah nama Anda pada daftar untuk melihat KTA Eksekutif (menampilkan Biodata, Pangkat, & XP).
  • Sistem XP & Pangkat: Membayar kas tepat waktu/di muka akan menaikkan XP. Menunggak akan memotong XP. Pangkat terendah (AFK) membuat KTA menyala merah.
  • Fire Streak (🔥): Menunjukkan kedisiplinan beruntun. Sistem memberikan Grace Period (Batas Toleransi) 1 bulan ke belakang sebelum streak hangus menjadi 0.
MENU PROGRAM

Fungsi: Memantau khusus arus kas acara atau kepanitiaan.

  • Menampilkan target anggaran acara (misal: 17-an) dan progress bar serapan dana (Pemasukan vs Terpakai vs Sisa Akhir).
MENU SISTEM

Fungsi: Pusat bantuan, identitas sistem, dan pengaduan rahasia.

  • Profil Sistem: Menampilkan versi aplikasi. Jika Anda masuk sebagai "Warga RT 05", KTA Holografik tidak akan muncul di sini. KTA eksklusif hanya untuk "Anggota NAVASENA" terverifikasi.
  • Kotak Suara (Aspirasi): Tombol merah melayang di kiri bawah untuk mengirim pesan, kritik, atau temuan kejanggalan secara anonim langsung ke kotak Bendahara.
  • Ganti Profil / Keluar: Tombol untuk mereset sesi dan kembali ke menu pemilihan profil awal.

OTORISASI SISTEM
